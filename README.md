## 方法

### getKey
返回对象:

```javascript
{
    domain:'m.baidu.com', // 来源域名，无值时返回 ''
    key:'怀孕', // 搜索词,无值时返回 ''
    referrer:'http://m.baidu.com/blabla' // referrer 值
}
```
